# Nx Experimental Project

This is an experimental project for testing out what Nx has to offer.

## Structure

The apps are lightweight and make use of shared libraries and shell libraries.

```folders
apps/
  angular/
    example-app/
  react/
  express/
  |
  |
  etc/
libs/
  shared/
    ui/
    data-access/
    util/
  example-app/
    feature/
      feature-1/
      feature-2/
      |
      |
      etc/
    ui/
    data-access/
    util/
   |
   |
   etc/
tools/
  tbd/
```
